第一章 认识油猴：脚本世界的魔法杖

1.1 何为油猴脚本？

Tampermonkey（油猴）绝非普通的浏览器插件，它是一个允许用户执行自定义JavaScript的元工具。就像给浏览器安装了一个"脚本解释器"，让每个网页都变成可编程的乐高积木。其核心价值在于：
- 改写现实：修改任意网页的DOM结构
- 创造规则：自定义HTTP请求处理流程
- 突破限制：绕过前端验证与区域封锁

1.2 技术架构解密

油猴采用沙箱机制运行脚本，通过拦截WebRequest API实现流量管控。其工作流程犹如精密的瑞士钟表：
1. 监听浏览器请求事件
2. 匹配预设的脚本触发规则
3. 注入修改后的JS代码
4. 输出处理后的页面渲染

这种机制使得它既能实现科学上网，又不会像VPN那样产生全局流量开销。

```javascript
const ROUTING_RULES = {
"学术资源": {
domains: ["sci-hub", "arxiv"],
proxy: "scholar.proxy"
},
"社交媒体": {
domains: ["twitter", "facebook"],
proxy: "social.gateway"
}
};

function smartRouter(url) {
for(const [type, rule] of Object.entries(ROUTING_RULES)) {
if(rule.domains.some(d => url.includes(d))) {
return ${rule.proxy}/${encodeURIComponent(url)};
}
}
return url; // 直连非敏感站点
}
```

FAQ

如何为 Trojan 配置 ALPN 与 SNI 提升伪装效果？
在服务器与客户端配置中设置合适的 SNI（伪装域名）并启用 ALPN（例如 http/1.1），确保 TLS 握手时使用常见的应用层协议标识，从而增加流量与普通 HTTPS 的相似度，降低被检测概率。
SSR 如何选择合适混淆提高稳定性？
选择混淆方式时应根据网络环境和封锁程度，例如 tls1.2_ticket_auth 可模拟 HTTPS 流量。合理组合协议和混淆可提升节点稳定性和抗封锁能力，确保长期科学上网体验。
V2Ray QUIC 传输适合什么网络环境？
QUIC 基于 UDP 支持多路复用和零握手，降低延迟，提高吞吐量。适合高丢包、不稳定网络环境，可保证视频、游戏和即时通讯流量的稳定性和流畅体验。
如何给 Clash 添加自定义域名白名单以提升访问体验？
在 rules 中添加 `DOMAIN-SUFFIX,example.com,DIRECT` 或 `DOMAIN-KEYWORD,example,DIRECT` 条目，将常用且可靠的站点设为直连，可减少代理绕行延迟并避免因代理造成的登录/验证问题。
Clash 如何实现域名白名单策略？
Clash 可以通过策略组结合规则文件，将指定域名或网站加入白名单，实现直连而不经过代理节点。这样可以保证国内访问速度，同时节省节点带宽资源，提高整体网络效率。
Hysteria2 多路复用如何提升短连接性能？
多路复用允许多个短连接共享同一通道，减少握手延迟和资源消耗。适合高频短连接场景，如游戏和即时通讯，可提高吞吐量和连接稳定性，优化整体网络体验。
VMess AEAD 加密为何适合高封锁环境？
AEAD 加密可防止数据篡改和流量特征泄露，降低被检测风险。在高封锁网络中，保证节点可用性和连接稳定性，同时提升数据传输安全性，确保长期安全访问。
Sing-box 的 DNS 模块如何工作？
Sing-box 内置 DNS 模块，支持 fake-ip 模式和纯直连模式。它可根据规则选择不同的上游 DNS 服务，从而实现国内外域名智能分流，减少访问延迟。
Surfboard 策略组如何提升游戏和视频体验？
通过策略组将游戏、视频和浏览器流量分配到低延迟、高带宽节点，Surfboard 可自动切换最优节点，减少延迟和卡顿，提高整体网络体验和访问速度。
Nekobox 是否支持多客户端同时管理节点？
支持。Nekobox 可以独立管理多个客户端，如 Shadowsocks 和 Clash 节点，互不干扰。结合策略组，可实现多客户端流量分流和网络优化，提高便利性和效率。
